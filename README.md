# Mafia Bot

Этот бот для игры в мафию для Telegram.

## Функциональность

- Присоединение к игре
- Начало игры
- Обработка ночных действий
- Голосование
- Проверка победителя

## Стек технологий

- Python
- pyTelegramBotAPI

## Запуск

Запустите бота:
```sh
python bot/bot.py\

Подключен к хостингу mafia300500.py в PythonAnywhere

Скриншоты
Скриншоты взаимодействий с ботом находятся в папке bin/.
